// ========================================
// CONNECTIONS (Fixed for final architecture)
// ========================================

// Frontend delivery
User > WAF: 1. Open web app
WAF > CF
CF > S3Web

// Authentication
User > Cognito: 2. Sign up / Sign in
Admin > Cognito: 2. Admin sign in

// Cognito trigger (final stack)
Cognito > LambdaAutoVerifySES: PostConfirmation
LambdaAutoVerifySES > SES: Verify user email identity

// API access (JWT)
User > APIG: 3. User APIs (JWT)
Admin > APIG: 3. Admin APIs (JWT + Admin group)

// Upload & Analysis Pipeline (PRESIGNED UPLOAD + EVENTBRIDGE + STEPF)
APIG > LambdaUpload: 4a. Get presigned PUT URL
LambdaUpload > User: 4b. Return uploadUrl + contractId
User > S3Doc: 4c. PUT PDF to S3 (presigned)
S3Doc > EventB: 5a. ObjectCreated (uploads/)
EventB > StepF: 5b. StartExecution

// Step Functions workflow (visual chain for clarity)
StepF > LambdaOCR: Step 1
LambdaOCR > Azure: Extract text
LambdaOCR > LambdaSanitize: Step 2
LambdaSanitize > LambdaAI: Step 3
LambdaAI > Bedrock: Analyze
LambdaAI > LambdaSave: Step 4
LambdaSave > DDB: Save results
LambdaSave > LambdaNotify: Step 5
LambdaNotify > SES: Send email
StepF > LambdaError: On error

// User endpoints (complete)
